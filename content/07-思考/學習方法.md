# 目標

> 如果我們覺得某個模式用起來很順手，並不代表它就是有效的、能帶領我們走向目標的；因此，我們可能得嘗試另一種模式才能知道。在無法得知其他模式對我們效率的影響下，我們將不得不嘗試所有其他模式，這過程可能非常耗時。

> 其中一個關鍵在於，透過避免自己的「思考」（或試圖獨立解決任務），我們剝奪了自己在大腦中建立關聯鏈的機會。這些關聯鏈是我們腦細胞間的連結，服務於我們的思維過程和信息關聯，我們藉此學會解決問題或應對情況。因此，簡單來說，我們剝奪了自己學習新事物和進一步發展的能力。

> 如果有人聲稱他的方法是唯一正確的方法，且這方法對他有效，那他就應該堅持下去。然而，聲稱沒有其他方法，或許對那個人來說是對的，但這並不意味著其他人無法做到。舉個比喻來說，如果那個人不知道如何使用梯子，他們就只會在外部幫助下才能到達更高的樓層。
> 
> 無論我們心中有什麼目標，都必須下定決心。

# 決策過程

> 決策心理學的研究證實，人們的行為絕非僅基於成本效益考量。由於許多理性模型未能真實反映實際決策行為，個人的決策過程可理解為理性僅在有限範圍內發揮作用的歷程。

> 《社會與人格心理學指南》的研究人員批評電車難題這個情境過於極端。如果我們持續陷入這種處境，決策將會變得異常棘手。研究人員最終指出，這個範例之所以過度極端，在於人們無法被期待做出這類抉擇，且基本上要做出"正確"決定是"不可能"的。這項論述背後存在諸多不同的緣由。
> 
> 1. 對於做出決定的那個人來說，這在決策前、決策中及決策後都是一種巨大的心理負擔。
> 2. 提出這種說法的人們並未看到此一困境的解決方案。
> 3. 以及其他許多基於個人經驗的理由。

> 如果有人告訴我們某件事是不可能的，這個人不可能完全了解我們所連結的所有因素。只因為對方相信自己看不到完成某項任務或達成某個目標的解決方法，這僅僅表示對方沒有看到解決方案或實現目標的途徑。許多人接受了這種說法，卻將這種態度轉移到自身身上，導致他們甚至還沒嘗試，就已經放棄了渴望的目標或嘗試的念頭。結果，人們在尚未開始嘗試之前就已經失敗了。

> 倘若我們在某種情況中不知該如何行動，原因通常是我們尚未蒐集足夠的事實來做出明智的決定。然而，一旦我們掌握了充足的資料，就能更精確地預測最終可能獲得的結果，並基於所收集的事實，開拓出我們能夠採取的可能性與途徑。
> 
> 最後，這意味著只要做到以下幾點，就沒有人能質疑你的成功：
> 
> 決定（ `Decision Making` ）好你發自內心（ `Willingness` ）真正渴望達成、且能讓你有意識及潛意識都感到快樂（ `The Brain` ）的具體目標（ `The Goal` ）。

# 問問題

- 問題的「狀態」並不會影響問題的答案，因此問問題的好與壞只是徒勞
- 問題才是指引你前進的羅盤。

不要忘記：
- Pareto Principle
- Occam's Razor

A question is a sentence worded or expressed to elicit information. 假設我們看到`主機 A` 和`主機 B`。為此，我們可以提出以下問題——這也是我們在滲透測試中常問的：How is Host A connected to Host B? 我們原本希望透過提問來獲取資訊，但從這個問題中我們真的獲得了任何資訊嗎？──沒有。所以連這個定義本身也是錯的，那究竟怎樣才是對問題的精確定義？

問問題是要不斷搞清楚你、目標、已知、未知之間的關係。

- 我們使用 Windows 的目的是什麼？
- Windows 如何影響我們在其立場下的狀態？
- Windows 必須提供或執行哪些功能，才能透過遠端存取方式進行管理？
- 這些遠端存取方法如何影響 Windows，從而改變 Windows 的狀態？這些方法為我們提供了什麼？
- 遠端存取方法的目的為何？
- 我們所知的不同遠端存取方法如何影響我們？
- Windows 必須運行哪些服務，才能讓我們使用我們尚不了解的方法？
- 遠程存取方法是如何影響 Windows，從而改變其狀態的？這些方法又能為我們提供什麼？

> A right question is a precise question that allows us to establish the relationships between the components, to understand them, and to take us one step further to the required answer.

# 挫折

當我們無法自我肯定時，便會轉而向外尋求他人的認可。然而，若非同行者，無人能真正給予我們確切的肯定。為釐清這點，讓我們再次回到登山的比喻。

假設你已下山，走過漫長艱辛的路途。途中經過數個歇腳處與城鎮，數週後遇見另一人，你向他詢問自己是否表現良好。

此人若未與我們同行，便永遠無法評斷我們的表現好壞。以比喻來說，即使此人曾走過相同的路徑，所有環境因素也不可能完全一致——諸如降雨、雷暴、烈日、強風等。
