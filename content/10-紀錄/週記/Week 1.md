# 開始寫週記

受許多啟發，於是就決定開始寫週記，今天是第一個禮拜。要寫很多嘛？不要啦。就當做流水帳就好了。隨便紀一點都行，就算只是三句話，那也是個文章。

一開始都不要完美，先求有就好。然後我預計以後每個禮拜日晚上，會花幾個小時寫週記。大概就平日想到什麼就寫什麼，不管架構、不管文筆、不管流暢度，就無腦的寫，等到禮拜日晚上再統一整理他們。

原本是用這種方法寫日記，但啟動成本實在太高了，沒辦法每天，所以就改成週記。

# 期末考周
這裡禮拜就是期末周！

- 作業系統期末考
- 計算機網路概論期末考
- 程式語言概論期末考
- 英文期末報告的演講稿
- 軟體工程的報告影片（晚上）
- 自我察覺與人際互動的期末

其實所有期末考都超級簡單，看個考古題就可以直接去考了。我都把它們想的都太難了。英文期末報告完了，教授說我和 Monica 都做的超棒！還有軟工也順利結束了，那個文件重構和測試的地方還差點開天窗，還好我有發現，超虧賊。

我總算克服我大一英文報告的心魔了啊，我還記得我那時上台腦袋一片空白，早上背的台詞在台上全忘光，七分鐘只擠出了短短幾句話。但這次不一樣，我花了兩天準備了簡報和講稿，在台上講的超級順，教授甚至是下課單獨把我們叫下來表揚我們。幹，一切終於都有回報的感覺。

我有好多目標都是只要時間一到，其實做完根本就無所謂的東西，包括這個也是。我平時不會為了考試而學習，就是單純靠自己好奇心驅使，想看這領域的什麼就去學什麼。可以自己寫一個 Memory Allocator、自己做一個 Proxy Server。但在考試前你就是得這麼做，你就是得精通排程演算法、得背誦你這輩子大概不會再碰到的 Protocol。

但也不能因為考試考過了就放過自己，沒有硬實力的都不能叫做本科生，如果現在都忘記了，那你曾經微積分拿 A+ 又有何用？

# 第一次世界大戰
看了[大时叔叔](https://www.youtube.com/@spacenw/videos)一系列的一戰講解，做了一點筆記。

## 歷史

當時一戰的源頭，其實就是新興的國家發現資源都被前人佔光了，而歐洲近乎所有國家都在互相牽制、為了恐懼互相結盟，緊繃到一個極點。而這個大公刺殺事件，事實上只是一個「小事件」，但就是這個事件，順著整個引線，引爆了整個歐洲。而且大家當時還以為，這只是打幾個月就可以快速結束的戰役。

德國打算速戰速決，提出了瘋狂的施里芬計畫。六週內把法國滅掉，其他國家只能認了也來不及，之後再趕回去面對東方俄國的那群草台班子。但六週對於當時的德國士兵根本就是強人所難，相當於每天都要背著三十公斤的裝備跑全碼馬拉松，還只能三天吃一餐，水還只能喝路邊的髒水。三十幾天下來，已然成了一群喪屍軍隊。

這時東邊德國指揮官克魯克異想天開，認為不需按照原計畫包抄，可以直接突襲法國取勝。結果造成巨大破口，被法國抓住機會，這個喪屍軍團就此被徹底擊垮。最後雙方為了避免被包抄，「向海奔跑」，挖了 700 公里的超長壕溝。在這裡，面對機槍掃射，性價比最高的戰術就是挖戰壕。雙方就這麼僵持不下，德國被英國切斷供應鏈，迎來了「蕪菁冬天」，七十萬平民被餓死。德國無限制潛艇戰，擊沈盧西塔尼亞號，將美國拉入了戰爭。

聖誕節這天，「聖誕節停火」見證了戰場上的一絲人性，但也馬上被上層迅速修復，通過砲擊強迫恢復戰鬥狀態。

德軍提出放血計畫，預測法國絕對不會放棄易功難守的聖地凡爾登，對它狂轟濫炸，想讓所有法國人進去送死。而在法國的聖路與輪換制度的頑強抵抗下，最後居然還是 1:1 交換比。

英國為了幫助法國，採用好友營制度，征招兩百多萬志願兵。在戰場上，英國連續七天七夜在戰場上轟炸，以為砲擊可以清空鐵絲網，炸死所有德軍。結果德軍躲在地下 10 米的混凝土掩體，輕鬆收割所有「走上前送死」的英軍。結果第一天，英軍傷亡 5.7 萬人。英國無數小鎮在同一天收到了所有年輕人的死訊，社會結構瞬間崩塌。

當然，凡爾登戰役以及索姆河戰役，徹底擊潰當時所有國家的意志。1914 年，大家為榮耀而戰。1915 年，為了勝利而戰。可在 1916 以後，沒人知道為何而戰，只想結束這場惡夢。

截至今天，最後一集還上，所以還沒補完。
## 感想
這是我第一次，在 YT 上聽別人講歷史。

以前國中對歷史沒有多大興趣，因為我實在很難背起來各種詳細的資訊，因此甚至討厭歷史。這好像是我第一次聽長系列的歷史講解、沒有壓力的聽故事，還挺不錯的，果然我只是不喜歡被強迫背我不在乎的瑣碎東西而已。

歷史的教訓還真不是說假的。

- 因為不了解而輕忽（以為戰爭會打很快、英國把平民送上前線送死）
- 高估自己的實力（高估德軍的體力、意志力）
- 利用對方的弱點（凡爾登戰役）

以前的人相信這些苦難本身就是啟示，一切的意義於終點接會顯露。但在世界大戰後，這一切都顯的如此荒謬。我們站在上帝視角看，也會覺得他們的理性完全不理性，但當時誰知道呢？

# 翻譯外掛
最近很喜歡用[沈浸式翻譯](https://immersivetranslate.com/zh-TW/)這個外掛，因為家庭號訂閱超級便宜，所以一直不斷和朋友推薦希望他們入坑，就好像老鼠會成員一樣。

不過最後都沒有人入坑，不知道是我太不會推銷、太討厭、還是他們英文太好呢？

我自己是發現我閱讀速度太慢，有時甚至連理解句子架構都有困難，因此就要常常對照中文意思。我自己原先就有製作一個套件，一樣套了外部大語言模型 API，可以即時提供網頁的中英雙語翻譯。

不過用了沈浸式翻譯後，發現它完全屌打我的套件：

- 20M Tokens 算下來，訂閱的成本居然比套 DeekSeek V3 API 的價格還要低
- 相容做的比我的套件好
- 還有其他超多形式的翻譯。圖片、文件、漫畫、影片都可以 ...

於是我就果斷放棄，直接訂閱了。好不符合我資工的背景啊，照理來說我應該要鑽進這個兔子洞打敗它對吧。但為何這麼做呢？我其實根本不想做這種產品，我的生涯重點也不在這，也不想花更多的時間做一個別人的複製品，更甚至它的成本還比我低，那我何必呢？我有這時間還不如多打點 HackTheBox 的機器，看點自己喜歡的小說。

# 上頭
期末考完後，我們一群朋友出去吃飯。吃飯難免聊八卦，聊著聊著才發現我們都對某個朋友都「有點意見」。

他直言不諱，遇到不爽的就直接公審。這很多人都會，但他太容易上頭，一但他開始攻擊，他要馬就是一直緊抓不放，要馬就是容易波及到無辜的人。而最近，因為我們一起合作做小組專案，而這次專案正好就有類似的事發生，剛好以此為引爆點，結果接連爆出幾乎所有人之前與他合作時都有些「摩擦」。

我只能說他是真的高，他居然能做到，讓我們會為了怕被他公審而服從他的指令。好像若要成大事，有點偏執幾乎是肯定的。
